# 公文整合助手 - TRAE Skill

## 简介

公文整合助手是一个基于 TRAE 的 Skill，帮助机关单位、国企和事业单位的文秘人员将团队成员提交的各种格式素材，自动整理成符合 GB/T 9704-2012 标准的正式公文。

**核心原则：不改原文，只做格式对齐和内容重组。**

## 能力

### 1. 格式对齐
- 自动对齐素材的缩进、换行、标点符号
- 自动修正全角/半角标点混用
- 去除多余空格，符合公文排版规范
- 保留原文措辞，绝不改写内容

### 2. 智能内容重组
- 支持 10 种公文文种：汇报材料、报告、请示、通知、函、纪要、简报、讲话稿、工作总结、工作方案
- 基于关键词评分 + 位置特征 + 上下文感知，将零散素材分类到对应章节
- 自动过滤信件格式中的敬语和落款
- 后处理修正分类异常（开头段优先归为背景，结尾段优先归为计划）

### 3. 格式检测
- 自动检测数据点（金额、百分比、数量、日期等）并提示核实
- 检测政策依据引用（国发、国办发、中发等文件编号）
- 检测机构/会议名称
- 检测口语化表达（大幅扩充词库，覆盖 30+ 种常见口语化表达）
- 检测待核实标记
- 检测格式问题（标点空格、联系方式等）

### 4. 素材间重复检测
- 检测不同素材之间的重复/相似段落
- 使用最长公共子序列（LCS）算法计算相似度
- 相似度 &ge; 90% 标记为重复，&ge; 70% 标记为相似

### 5. 数据一致性检查
- 检测同一数据在不同素材中的数值是否矛盾
- 覆盖金额、百分比、数量等数据类型

### 6. 对话粘贴转换
- 从 AI 对话窗口复制的 Markdown 内容直接粘贴
- 自动识别标题、段落、列表、表格、代码块、引用块
- Markdown 表格转为公文标准三线表（自动推断列对齐方式）
- 行内格式转换（加粗、斜体、代码、链接）
- 一键转为素材进入整合流程

### 7. 导出
- A4 纸张预览（仿宋字体、标准页边距、首行缩进两字符）
- 导出 Word（.docx）
- 导出 PDF（依赖浏览器打印功能）

## 使用方式

1. 打开 app/index.html（直接用浏览器打开，无需服务器）
2. 在「素材整合」页面上传 txt/md/docx 素材
3. 选择目标文种，系统自动对齐格式并重组章节
4. 查看格式检测报告，确认无问题后合并预览
5. 导出为 Word 或 PDF

## 技术栈

- 纯前端 HTML/CSS/JS，零后端依赖
- 所有文件处理在浏览器本地完成，数据不出本机
- JSZip 解析 .docx 文件
- FileSaver.js 导出文件
- docx.js 生成 Word 文档
