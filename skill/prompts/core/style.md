# 核心规则：语言风格与格式规范

## 1. 标题层级规范

公文及正式材料的标题层级必须严格遵循以下顺序，**不可跳级、不可混用**：

```
第一层：一、二、三、……（黑体，用于章/大段划分）
    ↓
第二层：（一）（二）（三）……（楷体，用于节/段内划分）
    ↓
第三层：1. 2. 3. ……（仿宋，用于条目列举）
    ↓
第四层：（1）（2）（3）……（仿宋，用于细目列举）
    ↓
第五层：① ② ③ …… 或 a. b. c. ……（仿宋，极少使用，仅在必要时）
```

### 1.1 层级使用规则
- **同一层级标题的语法结构应保持一致**。如第一层均使用「关于……」「加强……」等动宾结构，则全文统一。
- **标题末尾不加标点符号**（问号、叹号除外）。
- **标题中不使用序号重复**。如第一层已用「一、」，其下第二层从「（一）」开始，不重复使用「一」。

### 1.2 层级与字体的对应
| 层级 | 序号样式 | 字体 | 字号 | 排版 |
|------|----------|------|------|------|
| 第一层 | 一、 | 黑体 | 三号 | 左对齐，前空一行 |
| 第二层 | （一） | 楷体 | 三号 | 左对齐，前空半行 |
| 第三层 | 1. | 仿宋 | 三号 | 左对齐，缩进两字符 |
| 第四层 | （1） | 仿宋 | 三号 | 左对齐，缩进两字符 |

## 2. 正文格式规范

### 2.1 基本字体与字号
- **正文**：三号仿宋（GB2312 或方正仿宋），行距固定值 28-30 磅。
- **首行缩进**：2 字符（约 0.74 厘米）。
- **段落间距**：段前 0 行，段后 0 行，以行距控制疏密。
- **对齐方式**：两端对齐。

### 2.2 数字与标点
- **公历日期**：使用阿拉伯数字，如 `2024年7月4日`，不用「二〇二四年」。
- **文中数字**：除成文日期、部分结构层次序数外，一律使用阿拉伯数字。
- **结构层次序数**：第一层「一、」用汉字，第二层「（一）」用汉字加括号，第三层「1.」用阿拉伯数字。
- **标点符号**：使用全角中文标点。连接号用「—」（一字线），范围号用「～」。
- **引号**：使用「」和『』，不使用英文引号。
- **书名号**：使用《》和〈〉。

### 2.3 特殊元素
- **附件说明**：在正文下空一行左空二字编排「附件」二字，后标全角冒号和附件名称。如：
  ```
  附件：1.XX工作领导小组成员名单
       2.XX工作推进计划表
  ```
- **表格**：表头用黑体或加粗，表内用仿宋。表格上方应标注表序和表题，如「表1  2024年上半年工作进度表」。

## 3. 各级标题字体

### 3.1 公文标准字体
| 元素 | 字体 | 字号 | 效果 |
|------|------|------|------|
| 大标题（发文机关+事由+文种） | 二号小标宋体 | 22pt | 红色分割线下方居中 |
| 一级标题（正文内） | 黑体 | 三号 | 醒目 |
| 二级标题 | 楷体 | 三号 | 区别于正文 |
| 三级及以下标题 | 仿宋 | 三号 | 可加粗以示区分 |
| 正文 | 仿宋 | 三号 | 常规 |
| 发文字号 | 仿宋 | 三号 | 红色分割线上方居中 |
| 签发人 | 楷体 | 三号 | 右空一字 |

### 3.2 字体使用纪律
- 全篇**不得使用**艺术字、彩色字、下划线（除超链接外）、阴影、倾斜（外语除外）。
- 标题字体严格按层级对应，不得随意混用。
- 英文和阿拉伯数字的字体应与中文协调，推荐使用 Times New Roman。

## 4. 结尾用语规范

不同文种的结尾用语有严格规范，**不得混用、不得遗漏**。

| 文种 | 标准结尾用语 | 适用场景 |
|------|-------------|----------|
| **通知** | 「特此通知。」 | 各类通知通用，正文结束后另起一行 |
| **请示** | 「妥否，请批示。」 / 「以上请示，请予审批。」 | 上行请示必备 |
| **报告** | 「特此报告。」 / 「以上报告，请审阅。」 | 上行报告通用 |
| **函**（去函） | 「特此函达，请予函复。」 / 「请予支持为盼。」 | 平行文种 |
| **函**（复函） | 「特此函复。」 | 答复对方来函 |
| **批复** | 「此复。」 / 「特此批复。」 | 答复下级请示 |
| **通报** | 一般无固定结尾用语，可用「特此通报。」 | 视情况使用 |
| **决定** | 一般无固定结尾用语 | — |
| **纪要** | 一般无固定结尾用语 | — |

### 4.1 结尾用语位置
- 结尾用语应在正文内容结束后，**另起一段**，左空两字书写。
- 如成文日期前有印章位置预留，结尾用语与落款之间应留适当空行。

## 5. 落款规范

### 5.1 落款组成
落款由两部分组成，右对齐编排：
1. **发文单位全称或规范化简称**
2. **成文日期**（中文年月日，用阿拉伯数字书写，如 `2024年7月4日`）

### 5.2 落款格式
```
                           XX市人民政府办公厅
                               2024年7月4日
```
- 成文日期右空四字编排，发文单位以成文日期为准居中编排。
- 成文日期应使用阿拉伯数字将年、月、日标全，年份应标全称，月、日不编虚位。

### 5.3 印章位置
- 印章应端正、居中下压发文单位署名和成文日期，使发文单位署名和成文日期居印章中心偏下位置。
- 印章顶端应上距正文（或附件说明）一行之内。

## 6. 主送机关、附件与版记

### 6.1 主送机关
- **位置**：标题下空一行，左顶格编排。
- **全称或规范化简称**：使用主送机关全称、规范化简称或同类型机关统称。
- **多个主送机关**：按级别高低或类别排列，同级同类用顿号，不同类用逗号，末尾标全角冒号。
  - 例：`各省、自治区、直辖市人民政府，国务院各部委、各直属机构：`
- **普发性下行文**：主送机关过多导致首页无法显示正文时，移至版记中标注。

### 6.2 附件
- **附件说明**：正文下空一行左空二字标注「附件：」及附件名称。
- **附件本身**：另面编排，与正文一起装订时，「附件」二字及附件顺序号用三号黑体顶格编排在版心左上角第一行。
- **附件标题**：居中编排在版心第三行。

### 6.3 版记（抄送栏）
- **位置**：公文末页下端，左右各空一字。
- **组成**：抄送机关、印发机关和印发日期。
- **分割线**：版记中各要素之间用与版心等宽的粗细分隔线隔开。
- **抄送机关**：左右各空一字，「抄送」二字后标全角冒号，冒号后标注抄送机关名称，回行时与冒号后的首字对齐，最后一个抄送机关名称后标句号。

## 7. 行文语言规范

### 7.1 基本要求
- **准确**：用词精确，不模棱两可；一义一词，避免歧义。
- **简明**：开门见山，直述不曲；删繁就简，杜绝空话套话。
- **庄重**：用语规范，摒弃口语、网络语、方言、俗语。
- **得体**：符合文种特征和行文方向，上行谦恭、下行威严、平行协商。

### 7.2 禁用词汇
| 类型 | 禁用示例 | 替换建议 |
|------|----------|----------|
| 口语化 | 「这个」「那个」「说白了」「挺多的」 | 删除或改为规范表述 |
| 网络语 | 「点赞」「给力」「硬核」「破防」 | 删除或替换为标准词汇 |
| 模糊语 | 「大概」「可能」「差不多」「基本上」 | 提供精确表述或保留素材原貌 |
| 主观情绪 | 「令人痛心」「非常高兴」「极其恶劣」 | 除非素材原文，否则改为客观陈述 |
| 自我评价 | 「我们认为」「我方觉得」 | 视文种使用「经研究」「现决定」等 |

### 7.3 常用规范用语
| 场景 | 规范用语 |
|------|----------|
| 阐明依据 | 「根据……」「按照……」「依据……」 |
| 表明目的 | 「为了……」「为……」「旨在……」 |
| 说明经过 | 「经……研究」「经……同意」「经……批准」 |
| 提出要求 | 「请……」「务必……」「应当……」 |
| 表示态度 | 「同意……」「原则同意……」「不予同意……」 |
| 说明结果 | 「现将……通知如下」「现将……报告如下」 |
